🔑 Anthropic API 키 설정

AI 기능(사진 분석, 유사문제 생성)을 사용하려면 Anthropic API 키가 필요합니다.
키는 브라우저에만 저장되며 외부로 전송되지 않습니다.

API 키 발급: console.anthropic.com

☁️ Google Drive 연동 설정

데이터를 Google Drive에 저장하려면 Google Cloud Console에서
OAuth 클라이언트 ID를 발급받아 아래에 입력해 주세요.

📋 설정 방법 (5단계, 약 10분)
1. console.cloud.google.com 접속 → 새 프로젝트 생성
2. API 및 서비스 → 라이브러리 → Google Drive API 사용 설정
3. OAuth 동의 화면 → 외부 → 앱 이름 입력 후 저장
4. 사용자 인증 정보 → OAuth 클라이언트 ID → 웹 애플리케이션
5. 승인된 JS 출처에 http://localhost 추가 → 클라이언트 ID 복사
